# Holbertonschool-ChatGPT-Introduction

This repository is part of the **Holberton School curriculum** and focuses on learning how to use **ChatGPT as a debugging and development assistant**.

The project demonstrates how AI tools can help identify bugs, fix logical errors, improve input validation, and document code effectively across multiple programming tasks.

## ğŸ§© Project Content

The `debugging/` directory contains several scripts, including:

* **factorial.py**
  Fixes infinite loops, adds input validation, and documents functions.

* **print_arguments.py**
  Safely handles command-line arguments and execution permissions.

* **change_background.html**
  Fixes JavaScript event handling to dynamically change the page background.

* **mines.py**
  Implements a functional Minesweeper game with proper win detection.

* **checkbook.py**
  Adds error handling to prevent crashes from invalid user input.

* **tic_tac_toe.py**
  Fixes input validation, winner detection, and draw handling.
